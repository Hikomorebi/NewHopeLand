- 数据表名称
fdc.view_proj_equity_his

- 数据表解释

项目权益汇总表。

- 各字段说明

| 字段名      | 数据类型          | 描述     |
| ----------- | ----------------- | -------- |
| projcode    | character varying | 项目编码 |
| projname    | character varying | 项目名称 |
| equityratio | numeric           | 权益比例 |
| validdate   | character varying | 生效日期 |
| invaliddate | character varying | 失效日期 |
